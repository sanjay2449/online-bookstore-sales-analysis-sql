# 📖 Project Overview

This project simulates a real-world Online Bookstore Management System where customer purchases, products, suppliers, orders, shipments, payments, and reviews are stored in a relational database.

The project demonstrates the complete SQL workflow followed by Data Analysts—from designing the database to generating business insights and creating dashboard-ready views.

---

# 🎯 Objectives

- Design a normalized relational database
- Create tables with proper constraints
- Validate imported data
- Establish foreign key relationships
- Solve business problems using SQL
- Practice advanced SQL concepts
- Prepare SQL Views for Power BI Dashboard
